## 1. 画像と素材の管理 (`photo/`)

ポートフォリオで使用するすべての画像とPDFファイルは `photo/` フォルダに集約します。

### ✅ 運用のルール
| 項目 | ルール | 理由 |
| :--- | :--- | :--- |
| **ファイル名** | `half-size-alphabet.jpg` | 文字化けやリンク切れを防ぐため |
| **ファイルサイズ** | 1ファイル **2MB以下** | ページの読み込み速度を維持するため |
| **推奨形式** | `JPG` または `PNG` | 汎用性が高く、圧縮が効きやすいため |

> [!TIP]
> **画像の圧縮には [TinyPNG](https://tinypng.com/) などのツールを使うのがおすすめです。** ページの表示が驚くほど速くなります。

## 2. プロジェクトの編集と追加

`index.html` 内のプロジェクトセクションを編集する際のポイントです。

### 📌 新しいプロジェクトを追加する手順
1. **テンプレートのコピー**:
   `index.html` 内の `<!-- 📌 プロジェクト X -->` というコメントを目印に、ブロック全体をコピーします。
2. **貼り付けと番号更新**:
   既存のプロジェクトの後ろに貼り付け、コメントの番号を更新します。
3. **内容の書き換え**:
   `<h3>` (タイトル) や `<span class="tag-chip">` (技術) を編集します。

> [!IMPORTANT]
> **HTMLタグの閉じ忘れにご注意ください。** `<div>` を開いたら必ず `</div>` で閉じているか確認しましょう。

## 3. ロボットプロジェクト特記事項

`robot.html` で使用する特定の画像リストです。

| ファイル名 | 用途 |
| :--- | :--- |
| `robot-main.jpg` | メインのアイキャッチ画像 |
| `plate-before.jpg` | 改善前の比較写真 |
| `plate-after.jpg` | 改善後の比較写真 |
| `robot-presentation.pdf` | プレゼン資料の埋め込み |

## 4. トラブルシューティング

**Q: 画像が表示されなくなりました。**
- `photo/` フォルダの中に、指定したファイル名と**全く同じ名前**でファイルが存在するか確認してください（大文字・小文字も区別されます）。

**Q: 変更がブラウザに反映されません。**
- `Ctrl + F5` (Windows) または `Cmd + Shift + R` (Mac) を押して、強制的に再読み込みを試してください。
