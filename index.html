半自動メイクブラシ洗浄ロボット 🤖

3Dプリント CAD (Onshape) Raspberry Pi Python ロボティクス

👤 Role

企画・設計から実装まで全工程、CADで3Dモデリング、3Dプリント、プロトタイプ制作、市場調査

📋 Background

目的：メイクブラシ洗浄の手間を自動化し、衛生的に保つ

課題：手作業での洗浄は時間がかかり、洗浄力も不十分。初期案（コップ上でふるだけ）では洗浄力が弱かった

解決方法：3Dプリントで凹凸プレートを制作し、物理的にこする本当の洗浄方式を実現。外周ツメで回転防止、複数種類の凸で筆の内部まで洗浄可能に

📈 Result

  • • 3Dプリントで凹凸プレート（洗濯板）を設計・制作
  • • CADで一から作ったクマのデザインを装飾として配置
  • • 改良により板の回転を防止、洗浄力が大幅に向上
  • • 市場調査を実施し、差別化ポイント（物理洗浄×ロボット自動化）を明確化
  • • プロトタイプを複数回改良し、実用レベルの洗浄性能を達成

💭 Reflection

学んだこと：CADでの3Dモデリングスキル、プロトタイピングの重要性、ユーザー視点での問題発見と改良プロセス、市場調査の手法

改善点：完全自動化（Raspberry Piでモーター制御）、複数ブラシ同時洗浄機能、洗浄液の自動交換システムを追加したい
